Итак, когда выучены значения тригонометрических функций, можно перейти к уравнениям.


Перед этим стоит отметить, что для каждой тригонометрической функции существует обратная функция, которая имеет приставку arc.
То есть, sin(a) = b <=> arcsin(b) = a, cos(a) = b <=> arccos(b) = a, tg(a) = b <=> arctg(b) = a, ctg(a) = b <=> arcctg(b) = a
Где a - градусы (радианы), b - значение тригонометрической функции от -1 до 1.
Например, sin(45°) = √2/2, а arcsin(√2/2) = 45°. tg(π) = 0, a arctg(0) = π.

Свойства обратных функций:
arcsin(-a) = -arcsin(a), arccos(-a) = π - arccos(a), arctg(-a) = arctg(a), arcctg(-a) = arcctg(a)


Тригонометрические уравнения имеют вид: sin(x) = a.

Решения тригонометрических уравнений:
- - - - -
(+ 2πk - все обороты вокруг тригонометрической окружности, которых бесконечно много, ведь k - любое целое число)

Синус:
sin(x) = a
x1 = arcsin(a) + 2πk, k∈Z
x2 = π - arcsin(a) + 2πk, k∈Z

Косинус:
cos(x) = a
x = +-arccos(a) + 2πk, k∈Z

Тангенс (котангенс):
tg(x) = a   (ctg(x))
x = arctg(x) + πk, k∈Z   (arcctg(x))

Целые значения a:
    a	      -1		0		1
sin(x)	  -π/2 + 2πk, k∈Z     πk, k∈Z	  π/2 + 2πk, k∈Z
cos(x)	    π + 2πk, k∈Z   π/2 + πk, k∈Z     2πk, k∈Z
